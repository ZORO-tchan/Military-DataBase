# نظام إدارة المركز الثاني للمستجدين — V3
### الإدارة العامة لتدريب قوات الأمن

برنامج سطح مكتب (Python + Tkinter) لإدارة المجندين، الإجازات، الغياب، العودات،
والصرفيات — مع تقارير PDF عربية حقيقية، استيراد/تصدير Excel، تسجيل دخول، ونسخ احتياطي تلقائي.

مصمَّم ليعمل على أجهزة ضعيفة (**Windows 7 / Core 2 Duo / 3GB RAM**) باستخدام
**SQLite** كقاعدة بيانات أساسية (سريعة وخفيفة وبدون خادم).

## 🚀 التشغيل

### 1) تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

### 2) التشغيل
```bash
python -m markaz2.main
# أو من داخل المجلد:
python run.py
```

### أول تشغيل
- يُنشأ مستخدم افتراضي: **admin / admin** (غيّر كلمة المرور لاحقاً).
- تُنشأ دفعة افتراضية "يناير 2026".

## 📥 استيراد Excel

- يدعم الصيغ: **`.xlsx` / `.xlsm` / `.xls`** (الملفات القديمة Excel 97-2003 / WPS).
- استخدم زر **"قالب استيراد"** لإنشاء ملف قالب بالأعمدة الصحيحة.
- الأعمدة المدعومة: رقم الشرطة، الاسم، المحافظة، المؤهل، دفعة، العهدة،
  الماهية، رقم الهاتف، السرية، تاريخ الحضور، الديانة، صنعة-مهنة، تليفون الأب، ملاحظات.
- العمودان **رقم الشرطة + الاسم** إلزاميان.
- يبحث البرنامج عن صف العناوين تلقائياً خلال أول 15 صفاً (يتخطّى الشعار/الترويسة).
- الأرقام المكررة تُتخطّى تلقائياً (مع تقرير بالنتيجة).

> ملاحظة: الملف الأصلي `يناير 2026.csv` هو فعلياً `.xls`؛ غيّر امتداده إلى `.xls`
> ثم استورده مباشرة من زر "استيراد Excel".
