# 📥 طريقة الحصول على ملف EXE جاهز من المتصفح (بدون أي برامج على جهازك)

سنستخدم **GitHub Actions** — يبني الـ EXE لك في السحابة (مجاناً) وتحمّله جاهزاً.
كل اللي تحتاجه: **حساب GitHub مجاني + متصفح**. خلاص.

---

## الخطوات بالترتيب

### 1) أنشئ حساب GitHub (لو مالكش)
ادخل https://github.com واعمل Sign up (مجاني).

### 2) أنشئ مستودع (Repository) جديد
- اضغط زر **+** فوق على اليمين ← **New repository**.
- اكتب أي اسم مثل: `markaz2`
- اختر **Public** (أو Private، الاثنان يعملان).
- اضغط **Create repository**.

### 3) ارفع الملفات
في صفحة المستودع الجديد:
- اضغط **uploading an existing file** (أو Add file ← Upload files).
- **فك ضغط** ملف `markaz2_full.zip` على جهازك أولاً.
- اسحب **كل المحتويات** (مجلد `markaz2` + ملف `build_exe.bat` + مجلد `.github`)
  وأفلتها في صفحة الرفع.
  > ⚠️ مهم جداً: لازم يكون مجلد **`.github`** موجوداً ضمن المرفوعات
  > (هو اللي فيه أمر البناء). لو الويندوز بيخفيه، فعّل إظهار الملفات المخفية.
- اضغط **Commit changes** بالأسفل.

### 4) شغّل البناء
- ادخل تبويب **Actions** (فوق في صفحة المستودع).
- لو ظهرت رسالة تطلب تفعيل، اضغط **I understand my workflows, enable them**.
- على اليسار اختر **"بناء EXE لويندوز"** ← على اليمين اضغط **Run workflow** ← **Run workflow**.
- (أو سيبدأ تلقائياً بمجرد رفع الملفات).

### 5) انتظر دقيقتين ثم حمّل الـ EXE
- اضغط على عملية البناء الجارية (دائرة صفراء تتحول لعلامة ✅ خضراء).
- انزل لأسفل لقسم **Artifacts**.
- اضغط على **"المركز-الثاني-EXE-ويندوز"** لتحميل ملف ZIP.
- بداخله: **`المركز_الثاني.exe`** — جاهز للتشغيل بالدبل كليك. 🎉

بيانات الدخول الأولى: **admin / admin**

---

## ❓ ليه الـ CMD كان بيفتح ويقفل بسرعة عندك؟
السبب غالباً أحد اثنين:
1. **Python غير مثبّت** على جهازك (أو مش مضاف لـ PATH) — فالأمر يفشل فوراً.
2. كان فيه خطأ استيراد في طريقة البناء القديمة (تم إصلاحه الآن ✅).

مع طريقة GitHub Actions أعلاه، **مش محتاج Python ولا أي تثبيت** — الملف الناتج
EXE مستقل بذاته يشتغل على أي ويندوز مباشرة.

---

## بديل: البناء على جهازك (لو عندك Python)
لو حابب تبنيه محلياً بدل المتصفح:
1. ثبّت Python من https://www.python.org/downloads (فعّل Add Python to PATH).
2. ضع `build_exe.bat` بجوار مجلد `markaz2`.
3. دبل كليك على `build_exe.bat` وانتظر.
4. الملف الجاهز: `dist\المركز_الثاني.exe`
